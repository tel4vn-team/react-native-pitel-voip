# Installation

1. npm install -g yalc
2. In react-native-pitel-voip

```js
yalc publish
```

3. In rn-pitel-demo

```js
npm uninstall -S react-native-pitel-voip
yarn add react-native-pitel-voip
yarn i
```

# Update package

1. In react-native-pitel-voip

```js
yalc push
yalc push --script
```

2. In rn-pitel-demo

```js
yarn i
```
